    1. how much time do I take to complete task x?

    requirement: literate programming using org-babel to tangle codes in non linear ways. This idea can be expanded to utilize leo ideo ideas of “outline nodes organization” (finer than file-based organization). To do this each org-babel block must be able to references “inside” other blocks (not only as input to other block) by tagging nodes of the block. t

    Demo of this feature is demonstrated by using org-babel with org block and org-roam literater devops + nodes-outlines organization (from leo IDE).
